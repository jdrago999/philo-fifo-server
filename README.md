## Docker environment setup

Turn the thing on with:

```bash
docker-compose up
```

Enter the container with:

```bash
docker exec -it philo_server_1 bash
```

Start the server with:

```bash
docker exec -it philo_server_1 ruby server.rb
```

Run test suite with:

```bash
docker exec -it philo_server_1 ruby stack-test.rb
```
